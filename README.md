简介
--
校园达人是为高校学生量身制作的在线校园社交平台，通过手机App、微信公众平台和Web站点为学生提供学习、生活、社交等信息化服务。其中为方便广大新生入学，减少更多不必要的麻烦，校园达人提供了一个入学指引、玩转校园指南类的App；此外，还推出了一系列包括考试资料查询共享与下载、知识问答、经验在线交流等学习类的实用型App，比如自习室查询助手、考试助手、校园学习考试资料共享App等；还拥有课表秘书功能，规定时间段提醒上课时间、地点，同时还可以与上同一节课及同院系同学进行聊天；图书馆助手查询图书，室内找书指引，超期提醒，一键续借，扫一扫荐购；并且针对学生们生活中普遍关注的美食出行、兼职求职、物品交换、电脑维护等来提供资讯推荐，比如校园宣传栏App（可发广告、失物招领等）、二手物品交易App、腾讯QQ助手等；校园达人更是以社团组织、班级活动的动态跟踪为主，将各类集体生活的多彩斑斓映射到校园的每个角落，如校园热点新闻、校园BBS，校园贴吧等。Web端访问地址->http://xlanlab.com

安装配置
----

**android安装配置**

1. git clone源码到本地；
2. 导入App文件夹下项目到已经配置好Android开发环境的Eclipse中；



**server端安装配置**

1. git clone源码到本地；
2. 复制camsns文件夹到php服务器下，导入根目录下的最新数据库文件，配置config目录下的config.inc.php数据库连接信息;[在线源码部署地址](http://xlanlab.com)
